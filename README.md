# Steam Game Launcher

This is a very simple `windows-only` executable to launch steam games through the command line. Made to be mainly used by other applications. For example: launching this app with a steam game ID through your IDE to test a mod for a game.

## Usage
Create a shortcut and paste the steam id after the target.

or

Launch the terminal and paste the following command (make sure you're in the same directory as the executable):

`SteamGameLauncher.exe <AppID> [-D]`
- AppID = steam ID of the application
- -D = Debug flag to keep the console open
